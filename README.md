# 📧 Email Spam Classifier 🚫✅

A machine learning-powered web application that detects whether an email is **Spam** or **Not Spam** using Natural Language Processing and advanced text analysis algorithms.

## 🌐 Live Demo
[View Live Application](https://dazzling-frangipane-711255.netlify.app)

## 🔍 Features
- **Real-time Analysis**: Instant spam detection as you type
- **Advanced NLP**: Uses TF-IDF vectorization and pattern recognition
- **Interactive UI**: Beautiful, responsive design with detailed insights
- **Sample Emails**: Pre-loaded examples to test the classifier
- **Detailed Reports**: Shows spam indicators, confidence scores, and risk factors
- **Production Ready**: Fully optimized and deployed

## 🛠️ Technology Stack
- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Deployment**: Netlify

## 🚀 Getting Started

### Prerequisites
- Node.js 16+ 
- npm or yarn

### Installation
```bash
# Clone the repository
git clone https://github.com/yourusername/Email-Spam-Classifier.git

# Navigate to project directory
cd Email-Spam-Classifier

# Install dependencies
npm install

# Start development server
npm run dev

🧠 How It Works
The spam classifier uses several techniques:

Keyword Analysis: Detects spam-related terms and phrases
Pattern Recognition: Identifies suspicious patterns like excessive caps, multiple exclamation marks
Content Analysis: Analyzes email structure and formatting
Risk Assessment: Evaluates multiple factors to generate confidence scores
